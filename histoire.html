L'histoire de my Learning :

Lancer en 2020 My Learning est à l'origine un site pour illustrer un projet de fin d'anneé crée par deux étudiants dans le but d'aider les étudiants dans leur devoirs.


Ce site a été réalisé avec les langages de programmation suivants :

- html
- javascript
- css




My Learning est une plateforme internet visant à aider les élèves dans le besoin de ressources complémentaires, regroupent les différentes matières du tronc commun et des options de spécialité.
